Slider

A form control which lets the user specify a numeric value between a given range. Common use cases include configuring settings such as audio volume, color configuration, game difficulty etc.

Default slider

Default slider

Specifying minimum and maximum values

By default, the minimum is 0 and the maximum is 100. You can specify different bounds using the min and/or max attributes.

Setting granularity

By default, the value granularity is 1. Change the step attribute to control the granularity. For example, If you need a value between 0 and 100, accurate to 10% increments, you should set step="10".

If you need more granular control, the step attribute also accepts decimals.

If you want to accept any value regardless of how many decimal places it extends to, specify a value of any for the step attribute.

Disabled state

If a slider is currently unavailable for some reason, you can apply a disabled state using the is-disabled class and the disabled attribute.

Grid

A 12–column grid system for handling layout. The grid is responsive, mobile-first and built on CSS grid.

Basic grid structure

A grid consists of up to 12 .GridColumn elements wrapped in a .Grid element.

Equal-width columns

By default, grid columns have equal widths. Add any number of grid columns to a grid component and they will each assume an equal width.

Explicit-width columns

You can define an explicit width for each column by appending the desired column width to the GridColumn class. The combined column widths must total to 12.

Setting one column width

If you define an explicit width for just one column, the excess space will be automatically distributed between the other columns.

Responsive column count

You can define a different column count for each responsive breakpoint by using the 4 breakpoint class prefixes. Begin by defining the desired column count for mobile, then add classes for larger breakpoints as necessary.

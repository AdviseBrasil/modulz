List Item

Display rows of similar content inside a list. List Item components commonly appear inside Dropdown and SideDrawer components.

Basic structure

The only requirement is a series of <li class="ListItem"> elements wrapped in a <ul class="ListItemGroup">. Inside each <li>, you can add any layouts and components you like.

Interactive list items

Often, you will need a clickable List Item. You can do this by adding an <a> inside the <li> and moving the .ListItem class to the <a> instead. Add an additional interactive class to apply hover and active effects.

Don't put interactive components inside an interactive List Item.

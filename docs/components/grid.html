Grid

A 12—column grid system for handling layout. The grid is responsive, mobile-first and built on flexbox.

Basic grid structure

Grids are composed of a Grid container and any number of GridColumn children whose widths total to 12 columns.

6
6

Equal-width columns

By default, grid columns have equal widths. Add any number of grid columns to a grid component and they will each assume an equal width.

Explicit-width columns

You can define an explicit width for each column by appending the desired width to the GridColumn class. The combined column widths must total to 12.

Setting one column width

If you define an explicit width for just one column, the other columns will automatically divide the remaining space between themselves.

Multiple rows

Grid columns will automatically flow onto multiple rows. Just make sure the total column width per row adds up to 12.

Responsive column widths

You can define a different column width for each responsive breakpoint by using the 4 responsive class prefixes. Begin by defining the desired column width for mobile, then add classes for larger breakpoints as necessary.

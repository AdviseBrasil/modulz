Context

A contextual panel, overlaying the main content, which can house a variety of actionable components.

Markup structure

Firstly, you need a trigger component, which is usually a Button. Wrap your trigger component and <div class="Context"> inside <div class="ContextContainer js-ContextContainer">. When .ContextContainer js-ContextContainer is clicked, an is-active class will be added, which activates the Context panel.

You can also use a Link as your trigger component. The caret will always be centered on its x-axis and your Context panel will be positioned automatically.

Adding content

There are no limitations restricting which components can be placed inside a Context component. Child components do not receive any style modifications when placed inside a Context component.

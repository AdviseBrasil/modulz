Getting started

Copy the line of code below and paste it in the <head> of your html files.

<link rel="stylesheet" href="https://s3.com/modulz@0.5.0/css/modulz.min.css"/>

or install with npm

npm install modulz

or grab all the source files and build+develop locally

<link rel="stylesheet" href="https://s3.com/modulz@0.5.0/css/modulz.min.css"/>
